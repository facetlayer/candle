# list / ls

List services for the current project directory.

## Syntax

```bash
candle list
candle ls
```

## Description

The `list` command displays all services in the current project directory.

This includes:
 - Services that are actively running.
 - And services that are configured in .candle.json that aren't running.

## Examples

### List services in current project

```bash
candle list
```
